# E-commerce Store Database Project
Designed and developed a relational E-commerce Store Database using MySQL featuring normalized tables, foreign key relationships, advanced SQL queries, views, stored procedures, triggers, and business analytics operations for real-world order and inventory management.

## Project Overview

This project is a SQL-based E-commerce Store Database Management System developed using MySQL.

The system manages:

* Customers
* Products
* Orders
* Payments
* Reviews
* Product Variants

## Technologies Used

* MySQL
* MySQL Workbench
* SQL

## Features

* Database normalization
* Primary and foreign keys
* JOIN queries
* Aggregate queries
* Views
* Stored procedures
* Triggers

## Database Tables

1. Customers
2. Products
3. Categories
4. Orders
5. Order_Items
6. Payments
7. Reviews
8. Product_Variants

## Sample Queries

* Top selling products
* Revenue analysis
* Customer spending analysis
* Product reviews analysis

## Project Outcome

This project demonstrates practical database design and SQL query implementation for a real-world e-commerce system.
